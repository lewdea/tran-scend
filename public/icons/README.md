# 图标文件说明

这个目录需要包含以下尺寸的图标文件：

- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

你可以使用任何图标设计工具创建这些图标，或者使用在线工具如：
- [IconKitchen](https://icon.kitchen/)
- [Canva](https://www.canva.com/)
- [Figma](https://www.figma.com/)

建议设计一个包含 "翻译" 或 "地球" 元素的图标。

临时方案：在开发测试期间，扩展可以不需要图标文件也能正常运行。
